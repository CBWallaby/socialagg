# socialagg
A vibe-coded browser extension that aggregates social media feeds from open tabs

# Installation
This is not yet a packaged & signed extension - You will need to turn on "Developer Mode" in your browser's extension settings. You can then use "Load unpacked" to install the extension.

# Usage
1. Open the extension, it will start with no content.
2. Open a new tab and sign in to X, BlueSky, or a Mastondon host.
3. As content loads in your social media tabs, the content will begin aggregating in the "Unified Social Feed" tab.
4. Switch to the "Unified Social Feed" tab to enjoy all your posts in one feed.
5. Click on a post to be taken to the original.
